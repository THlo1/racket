## 使用场景
因安服需要进行应急演练及安全意识教育，不能对生产环境造成破坏，故使用模拟勒索进行演示，达到寓教于乐的效果

### 声明

本程序仅模拟勒索病毒环境，加密指定文件夹下的所有文件，在加密完成后显示勒索提示;提供IP及端口可进行反弹shell。 

仅供学习或用于安全意识培训提供参考，未经授权请勿进行破坏性测试，利用提供的信息造成的直接或间接损失，由使用者承担。
